# Exercise

主要使用go語言來練習一些常見的演算法或者一些簡單的測驗

## Directory Structure

```
exercise/
│
├── algorithms/                  # 演算法主目錄
│   ├── sorting/                 # 排序演算法
│   │   ├── quicksort/
│   │   ├── mergesort/
│   │   ├── bubblesort/
│   │   └── heapsort/
│   │
│   ├── search/                  # 搜索演算法
│   │   ├── binary_search/
│   │   └── linear_search/
│   │
│   ├── graph/                   # 圖形演算法
│   │   ├── dijkstra/
│   │   ├── bellman_ford/
│   │   └── floyd_warshall/
│   │
│   ├── dynamic_programming/     # 動態規劃
│   │   ├── fibonacci/
│   │   ├── knapsack/
│   │   └── longest_common_subsequence/
│   │
│   └── data_structures/         # 資料結構相關演算法
│       ├── binary_trees/
│       ├── linked_lists/
│       ├── stacks_and_queues/
│       └── hash_tables/
│
├── design_patterns/            # 設計模式
│   ├── creational/
│   ├── structural/
│   └── behavioral/
│
├── system_design/              # 系統設計
│   ├── scalability/
│   ├── reliability/
│   └── security/
│
├── databases/                  # 資料庫相關
│   ├── sql_queries/
│   ├── normalization/
│   └── indexing/
│
└── challenges/                 # 編程挑戰和練習題
├── easy/
├── medium/
└── hard/
```
